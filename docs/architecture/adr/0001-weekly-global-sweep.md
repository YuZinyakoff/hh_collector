# ADR-0001: Канонический режим сбора — weekly global sweep

## Статус
Accepted

## Контекст
Платформа должна длительно и воспроизводимо накапливать данные о вакансиях hh.ru.  
Нужен режим, который даёт хорошее покрытие рынка и остаётся эксплуатационно реалистичным на одной VPS.

## Решение
Основным режимом сбора принимается weekly global sweep поискового пространства hh.ru.

## Последствия
Плюсы:
- предсказуемый режим работы;
- воспроизводимый ритм наблюдений;
- хорошая совместимость с panel-like анализом.

Минусы:
- менее плотное наблюдение, чем при ежедневном full sweep;
- часть краткоживущих вакансий может не попасть в наблюдение.
