# ADR-0004: Raw API payload сохраняется

## Статус
Accepted

## Контекст
Для воспроизводимости, отладки и расследования инцидентов нужен доступ к исходным ответам API.

## Решение
Сырые ответы API сохраняются вместе с метаданными запроса.  
В MVP raw payload допускается хранить в PostgreSQL.

## Последствия
Плюсы:
- упрощается отладка и повторный парсинг;
- снижается риск необратимой потери контекста.

Минусы:
- растёт объём хранения;
- требуется retention/archiving policy.
