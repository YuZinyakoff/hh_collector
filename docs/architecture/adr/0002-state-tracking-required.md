# ADR-0002: История наблюдений вакансий обязательна

## Статус
Accepted

## Контекст
Только текущее состояние вакансий недостаточно для долгосрочного анализа и контроля качества сбора.

## Решение
Система обязана хранить:
- first_seen_at
- last_seen_at
- seen events
- aggregated current state

## Последствия
Плюсы:
- появляется история присутствия вакансии на рынке;
- можно отделять текущее состояние от исторических наблюдений.

Минусы:
- увеличивается объём хранения;
- усложняется модель данных.
