# ADR-0005: Развёртывание v1 — одна VPS и Docker Compose

## Статус
Accepted

## Контекст
На старте проект должен быть простым в деплое и сопровождении.

## Решение
Первая production-like версия разворачивается на одной VPS в Docker Compose.

## Последствия
Плюсы:
- низкая операционная сложность;
- быстрый старт;
- удобство локального повторения среды.

Минусы:
- ограниченный запас по отказоустойчивости;
- масштабирование в основном vertical-first.
